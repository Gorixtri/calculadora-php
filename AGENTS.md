### Convenciones y arquitectura
- El código fuente principal debe estar en español, incluyendo comentarios y nombres de variables/funciones.
- El historial de operaciones y la gestión de pestañas son funcionalidades clave.
- Sigue las mejores prácticas de PHP para organización de archivos y seguridad.

### Comandos útiles
- No hay scripts de build o test automatizados definidos. Ejecuta los archivos PHP directamente en un servidor local o entorno compatible.
